租车指南

一、企事业单位租车

您需要提供单位营业执照副本、法人代码证书、单位公章、单位介绍信、驾驶员的身份证、驾驶证及牡丹卡。

二、个人租车

⑴如您是本市人，您需有正式工作并提供本人身份证、驾驶证、城八区户口本原件及相应的保证金和租金。

⑵如您是外地人，需提供本人身份证、驾驶证、并提供您城八区朋友的户口本原件、身份证、并本人到合同签约地为您担保，及相应的保证金和租金。

三、个人租车

您需要提供企业税务登记证，法人代码证书、首席代表证或企业常驻代表机构登记证企业委托书、合同章，驾驶人的身份证、驾驶证及相应的保证金及租金。
